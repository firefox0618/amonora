### 🤖 Телеграм-боты 

**Назначение:** Основной интерфейс взаимодействия с пользователем и персоналом.

В системе работают **4 независимых бота**:

|Бот|Роль|Особенности|
|---|---|---|
|**@amonora_bot**|**Клиентский**|Основной продукт. Регистрация, триал, покупка, выбор страны, получение ключей, рефералка. Работает 24/7.|
|**@amonora_support_bot**|**Поддержка**|Принимает тикеты от клиентов. Операторы отвечают внутри бота. История переписки сохраняется.|
|**@amonora_control_bot**|**Админский**|Уведомления об оплатах, ошибках системы, алерты мониторинга. Кнопки быстрого управления (перезапуск сервисов).|
|**@test_amonora_bot**|**Тестовый**|Песочница. Используется разработчиками для проверки новых функций перед выкаткой на основных пользователей.|

- **Технологии:** Aiogram 3.x (асинхронная библиотека).
- **Принцип работы:** Боты не хранят состояние сами, они отправляют запросы в **Бэкенд**, который возвращает ответ. Это позволяет легко обновлять логику без перезапуска ботов.

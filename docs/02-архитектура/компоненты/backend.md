### 🧠 Бэкенд 

**Назначение:** «Мозг» системы. Обрабатывает всю бизнес-логику, хранит данные и управляет ботами и нодами.

- **Технологии:** Python 3.10+, FastAPI (асинхронный веб-фреймворк), SQLAlchemy (работа с БД), Pydantic (валидация данных).
- **Основные задачи:**
    - Прием запросов от Telegram-ботов и Dashboard.
    - Управление балансами пользователей и подписками.
    - Генерация конфигураций для VPN (ключей).
    - Взаимодействие с платежными шлюзами (обработка вебхуков).
    - Планирование фоновых задач (проверка истечения подписок).
- **Структура:**
    - `api/` — эндпоинты для панели и ботов.
    - `core/` — настройки, безопасность, зависимости.
    - `models/` — описание таблиц базы данных.
    - `services/` — бизнес-логика (оплаты, рефералы, генерация ключей).
    - `utils/` — вспомогательные функции.
- **Как запустить:** Через Docker Compose (`docker-compose up backend`) или локально с `.env`.

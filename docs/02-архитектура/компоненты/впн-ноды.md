### 🌍 ВПН-ноды

**Назначение:** Серверы, через которые проходит трафик пользователей.

- **География:**
    - 🇩🇪 **Германия:** Основная нода (высокая скорость).
    - 🇩🇰 **Дания:** Резервная/Альтернативная.
    - 🇪🇪 **Эстония:** Резерв на случай блокировок основных.
- **Программное обеспечение:**
    - **3x-ui:** Панель управления на самой ноде (для ручного контроля).
    - **Xray Core:** Ядро, обрабатывающее трафик (протоколы VLESS, VMess, Trojan).
- **Управление:**
    - Бэкенд подключается к нодам по **SSH** или через **локальное API** панели 3x-ui.
    - Автоматическое создание/удаление пользователей при покупке/отмене подписки.
    - Мониторинг нагрузки (CPU, RAM, Traffic) отправляется в Grafana.
- **Безопасность:** Доступ только по SSH-ключам. Порты панели закрыты фаерволом для внешнего мира.

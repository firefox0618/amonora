### 💻 Веб-панель 

**Назначение:** Графический интерфейс для администраторов и менеджеров (Dashboard).

- **Состав:**
    - **Frontend (`dashboard/ui`):** Next.js + React + TailwindCSS. Быстрый, адаптивный интерфейс.
    - **Backend (`dashboard/api`):** Отдельный слой API (или прокси к основному бэкенду) для выдачи данных панели.
- **Возможности:**
    - Просмотр списка всех пользователей и их статусов.
    - Финансы: история платежей, ручной начисление баланса.
    - Управление нодами: добавление серверов, просмотр нагрузки.
    - Настройка тарифов и промокодов.
    - Просмотр логов и тикетов поддержки.
- **Доступ:** Только по логину/паролю (JWT авторизация). Доступен из внутренней сети или через защищенный туннель.

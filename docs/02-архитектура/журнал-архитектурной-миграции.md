# Журнал архитектурной миграции

## Назначение

Этот файл фиксирует пошаговую архитектурную декомпозицию проекта.

Правило работы:

- перед каждым значимым шагом делается backup;
- после каждого шага фиксируется, что изменили;
- отдельно фиксируется, зачем это делали;
- отдельно фиксируются проверки;
- отдельно фиксируются последствия и риски;
- переход к следующему шагу делается только после проверки.

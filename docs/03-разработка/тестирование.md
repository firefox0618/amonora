# Тестирование Amonora

## Обзор

Тестовая поверхность проекта содержит **135+ файлов** в директории `tests/`. Тесты покрывают:
- Основной бот (обработчики, платежи, устройства, VPN)
- Dashboard (API v1 и v2, аутентификация, финансы, поддержка)
- Landing (вебхуки, публичные подписки, bridge access)
- Control Bot (диспетчер, уведомления, события)
- Crypto Pay, Platega, ручные платежи
- VPN провижининг (3x-ui, Xray Core)
- Аналитику, синхронизацию, региональную консистентность

---

## Как запустить ВСЕ тесты

```bash
cd /opt/amonora_bot
source venv/bin/activate
python -m pytest tests/ -v
```

С параллельным выполнением (если установлен pytest-xdist):
```bash
python -m pytest tests/ -v -n auto
```

С генерацией отчёта:
```bash
python -m pytest tests/ -v --tb=short --junitxml=test-results.xml
```

## Как писать новые тесты

### Шаблон теста

Создай файл `tests/test_<компонент>_<функция>.py`:

```python
"""Тесты для <краткое описание компонента>."""
import pytest
from unittest.mock import AsyncMock, patch, MagicMock

# Импорт тестируемых модулей
from bot.config import config
from bot.db import get_user_by_id
from backend.core.models import User


@pytest.mark.asyncio
async def test_<имя_сценария>():
    """Краткое описание что проверяет тест."""
    # Arrange — подготовка данных
    # ...

    # Act — вызов тестируемой функции
    result = await some_function(arg1, arg2)

    # Assert — проверка результата
    assert result is not None
    assert result.some_field == expected_value


@pytest.mark.asyncio
async def test_<имя_сценария>_with_mock():
    """Тест с мокированием внешних зависимостей."""
    with patch("bot.vpn_api.XUIClient") as MockClient:
        mock_client = AsyncMock()
        MockClient.return_value = mock_client
        mock_client.login.return_value = True

        # Act
        result = await function_using_xui()

        # Assert
        mock_client.login.assert_called_once()
        assert result is True


def test_<синхронный_тест>():
    """Синхронный тест (без async)."""
    from bot.utils.tariffs import get_tariff

    tariff = get_tariff("1m")
    assert tariff is not None
    assert tariff.code == "1m"
```

### Конвенции именования

| Паттерн | Описание | Пример |
|---------|----------|--------|
| `test_<функция>()` | Проверка одной функции | `test_get_tariff()` |
| `test_<функция>_with_<условие>()` | Функция в特定条件 | `test_get_tariff_with_invalid_code()` |
| `test_<сценарий>()` | Проверка бизнес-сценария | `test_subscribe_after_payment()` |
| `test_<компонент>_<поведение>()` | Поведение компонента | `test_dashboard_auth_lockout()` |

### Мокирование внешних сервисов

```python
# Мокирование Platega
with patch("bot.platega_flow.PlategaClient") as MockPlatega:
    mock_platega = AsyncMock()
    mock_platega.configured = True
    MockPlatega.return_value = mock_platega

    mock_platega.create_payment.return_value = PlategaPayment(
        transaction_id="tx_123",
        checkout_url="https://pay.platega.io/...",
        status="PENDING",
        payment_method_id=2,
        expires_in="3600",
        raw={},
    )

# Мокирование XUIClient
with patch("bot.vpn_api.XUIClient") as MockClient:
    mock = AsyncMock()
    mock.login.return_value = True
    mock.get_inbounds.return_value = {"success": True, "obj": []}
    MockClient.return_value = mock
```

## Запуск при CI/CD

Для полного прогона в CI:

```bash
python -m pytest tests/ \
  -v \
  --tb=short \
  --junitxml=test-results.xml \
  --cov=bot \
  --cov=dashboard \
  --cov=backend \
  --cov-report=xml \
  --cov-report=html
```

Для быстрого smoke-теста перед деплоем:

```bash
python -m pytest tests/ \
  -k "dashboard_auth or bot_start or db or crypto_pay" \
  -v --tb=short
```

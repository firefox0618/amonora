# Дополнительные слоты устройств

## Обзор

По умолчанию клиент получает **1 устройство** на подписку. Можно купить дополнительные слоты.

## Параметры

| Параметр | Значение | Конфиг |
|----------|----------|--------|
| Базовый лимит | 3 устройства | `DEFAULT_DEVICE_LIMIT` |
| Цена доп. слота | 49 ₽ | `DEVICE_SLOT_UNIT_PRICE_RUB` |
| Максимум доп. слотов | 5 | `DEVICE_SLOT_MAX_EXTRA_SLOTS` |
| Итого максимум | 8 устройств | базовый + 5 доп. |
| Лимит IP на клиент (3x-ui) | 1 | `VPN_MAX_DEVICES_PER_KEY` |

## Как работает покупка

1. Клиент нажимает «Купить слот» в экране «Мои устройства»
2. Выбирает способ оплаты (СБП, баланс, Stars)
3. После оплаты создаётся `DeviceSlotEntitlement`:
   - `slots_count = 1`
   - `unit_price_rub = 49`
   - `expires_at` = конец текущей подписки

## Лимит устройств

```
effective_limit = DEFAULT_DEVICE_LIMIT + sum(active entitlements)
```

Где `active entitlements` — записи в `device_slot_entitlements` со статусом `active` и `expires_at > now()`.

## Лимит IP для 3x-ui

Для Германии и Эстонии (3x-ui) лимит IP enforced через `limitIp` в панели. Для Дании (Xray Core) — через `xray_single_ip_enforcer.py`.

## Истечение слотов

При истечении подписки:
- `DeviceSlotEntitlement` деактивируется
- Существующие устройства **не удаляются** автоматически
- При возобновлении подписки — слоты восстанавливаются

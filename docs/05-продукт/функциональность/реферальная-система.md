# Реферальная система

## Как формируется реферальная ссылка

Каждому пользователю при регистрации генерируется уникальный реферальный код:

- **Длина:** 10 символов
- **Алфавит:** строчные буквы + цифры (`a-z`, `0-9`)
- **Формат ссылки:** `https://t.me/amonora_bot?start=ref_<code>`

При нажатии на ссылку новый пользователь попадает в бот с параметром `ref_<code>`, который сохраняется и привязывает приглашённого к рефереру.

**Источник:** `bot/utils/referrals.py` — `generate_ref_code()`, `build_referral_link()`
**Источник:** `bot/handlers/start.py` — `referral_token = command.args.split("_", 1)[1]`

## Когда начисляется бонус

Бонус начисляется **при первой оплате приглашённого пользователя**.

Процесс:
1. Приглашённый регистрируется по ссылке реферера
2. Приглашённый активирует trial
3. Приглашённый оплачивает подписку (любой тариф)
4. Система определяет реферера через `User.referred_by_user_id`
5. Бонус начисляется **обоим** — и рефереру, и приглашённому

Модель `ReferralReward` фиксирует каждую выплату:
- `referrer_user_id` — кто получил бонус
- `invited_user_id` — кого пригласили
- `payment_record_id` — за какую оплату
- `bonus_referrer_rub` — сумма рефереру
- `bonus_invited_rub` — сумма приглашённому
- `tariff_code` — за какой тариф

**Важно:** бонус начисляется только **один раз** за каждого приглашённого (при первой оплате). Повторные оплаты того же пользователя не приносят бонусов.

**Источник:** `backend/core/models.py` — `ReferralReward`, `bot/db.py` — `process_referral_reward_for_payment()`

## Размер бонуса

Бонус зависит от тарифа приглашённого:

| Тариф приглашённого | Бонус рефереру | Бонус приглашённому |
|--------------------|---------------|---------------------|
| 1 месяц (149 ₽) | 50 ₽ | 50 ₽ |
| 3 месяца (399 ₽) | 50 ₽ | 50 ₽ |
| 6 месяцев (749 ₽) | 50 ₽ | 50 ₽ |
| 12 месяцев (1390 ₽) | 100 ₽ | 100 ₽ |

Бонусы зачисляются на **внутренний баланс** (RUB) и могут быть использованы при оплате следующих подписок.

**Источник:** `bot/utils/referrals.py` — `REFERRAL_BONUS_BY_TARIFF`, `tests/test_referral_utils.py`

## Уровни реферальной программы

Реферальная программа имеет систему уровней, основанную на количестве **оплаченных** приглашённых:

| Уровень | Оплаченных рефералов | Прогресс-бар |
|---------|---------------------|-------------|
| Без уровня | 0 | `[░░░░░░░░░░]` |
| Новичок | 1–3 | `[███░░░░░░░]` |
| Продвинутый | 4–10 | `[███████░░░]` |
| Партнер | 11+ | `[██████████]` |

Прогресс-бар отображается в интерфейсе реферальной системы бота.

**Источник:** `bot/utils/referrals.py` — `calc_level()`, `render_progress_bar()`

## Ограничения

- **Один реферал на пользователя:** каждый пользователь может быть приглашён только один раз (`Referrals.invited_user_id` — unique constraint)
- **Один бонус за оплату:** `ReferralReward.payment_record_id` — unique constraint, предотвращает повторные выплаты за один платёж
- **Максимального количества рефералов нет** — пользователь может пригласить сколько угодно людей
- **Миграция баланса:** существовала миграция `referral_balance_migrated_at` для начисления бонусов за исторические приглашения (legacy cutoff)

**Источник:** `backend/core/models.py` — уникальные ограничения, `tests/test_referral_balance.py`

## Статистика рефералов

Пользователь видит в боте:

- **Реферальная ссылка** — для копирования и отправки
- **Баланс бонусов** (`balance_rub`) — доступные для списания рубли
- **Всего заработано** (`earned_total_rub`) — суммарный доход
- **Приглашено** (`invited_count`) — всего зарегистрировавшихся по ссылке
- **Оплатило** (`paid_count`) — из них совершили первую оплату
- **Текущий уровень** — название уровня
- **Следующий уровень** — сколько осталось до следующего
- **Прогресс-бар** — визуальная полоса прогресса

**Источник:** `bot/handlers/referrals.py` — `_show_referrals()`, `bot/db.py` — `get_user_referral_stats()`

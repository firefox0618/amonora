# Routing Packs (маршрутизация трафика)

## Обзор

Клиенты могут скачать **JSON routing packs** для VPN-клиентов, которые направляют:
- **Российские домены** → напрямую (без VPN)
- **Зарубежные домены** → через VPN

## Поддерживаемые клиенты

| Клиент | Платформа |
|--------|-----------|
| **v2rayNG** | Android |
| **Nekoray** | Windows / Linux |
| **Streisand** | iOS |

## Как работает

1. Клиент скачивает routing pack (JSON)
2. Импортирует в свой VPN-клиент
3. Клиент использует правила маршрутизации из JSON
4. Российский трафик идёт напрямую, остальной — через VPN

## Ограничение

URI-ключи (`vless://...`) **не могут** содержать правила маршрутизации. Поэтому routing pack — отдельный файл, а не часть ссылки.

## Формат

JSON-файл с правилами маршрутизации для соответствующего клиента. Содержит:
- Список российских доменов/IP (прямое подключение)
- Список зарубежных доменов/IP (через VPN)
- Настройки DNS

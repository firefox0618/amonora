## 💡 Как пользоваться этой документацией

1. **Определите свою роль** — выберите раздел выше, который соответствует вашей задаче
2. **Начните с рекомендованных файлов** — каждый раздел имеет файлы для быстрого старта
3. **Используйте поиск** — если знаете термин, найдите его в [Глоссарии](глоссарий.md)
4. **Проверяйте дату обновления** — актуальные файлы находятся в основных папках, старые — в [07-история/](07-история/)

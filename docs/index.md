# 🗺️ Карта документации Amonora

Главное меню — точка входа для любого члена команды. Ссылки сгруппированы по ролям.

> **Важно:** Основной бот имеет собственный `bot/router.py` и **не зависит** от тестового бота. Клиент **не выбирает** страну, протокол или режим — все серверы включены автоматически. Подробнее: [02-архитектура/потоки-данных/unified-subscription-flow.md](02-архитектура/потоки-данных/unified-subscription-flow.md).

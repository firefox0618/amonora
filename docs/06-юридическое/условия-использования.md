# Условия использования

**Amonora** (далее — «Сервис») предоставляет VPN-сервис через экосистему Telegram-ботов и веб-интерфейс. Используя Сервис, вы соглашаетесь с настоящими Условиями.

---

## 1. Описание Сервиса

Amonora — это Telegram-first VPN-сервис, предоставляющий:

- **Подключение** через протоколы VLESS и Trojan
- **Серверы**: Германия (3x-ui), Дания (Xray core), Эстония (резервный)
- **Режимы подключения**: Стабильный / Мобильный / Резерв
- **Клиентский путь**: через `@amonora_bot` с возможностью управления через веб-страницу `client.amonoraconnect.com/<token>`
- **Платформы**: Android, iOS, Windows, macOS, Linux, Apple TV, Android TV

---

## 2. Регистрация и аккаунт

### 2.1. Условия создания аккаунта

- Аккаунт создаётся автоматически при первом взаимодействии с `@amonora_bot`
- Требуется действующий аккаунт Telegram
- **Trial-период**: предоставляется один раз при подписке на канал

### 2.2. Устройства

- Пользователь может создавать, переименовывать и удалять устройства
- Каждое устройство привязывается к слоту (`1..N`)
- Server-side bind к `client.amonoraconnect.com` сохраняет метаданные устройства

### 2.3. Ответственность пользователя

- Пользователь несёт ответственность за сохранность своих ключей доступа (VLESS/Trojan)
- Запрещается передавать учётные данные третьим лицам

## 3. Тарифы и оплата

### 3.1. Способы оплаты

| Способ | Описание |
|--------|----------|
| **Telegram Stars** | Оплата через внутреннюю валюту Telegram |
| **Platega** | Автоматическая оплата через СБП или криптовалюту |
| **Внутренний Баланс (RUB)** | Авто-списание при оплате |
| **Ручная оплата** | Fallback-режим, обрабатывается администратором |

### 3.2. Реферальная программа

- Реферальные бонусы: 50 ₽ / 100 ₽ за 12 месяцев активности реферала
- ⚠️ Требуется уточнение: точные условия начисления

### 3.3. Промокоды и подарочные подписки

- Промокоды активируются через бота
- Подарочные подписки передаются через токенизированную ссылку

### 3.4. Возврат средств

Условия возврата описаны в отдельном документе: [Возврат и поддержка](возврат-и-поддержка.md).

## 4. Допустимое использование

### 4.1. Разрешено

- Личное некоммерческое использование
- Обход географических ограничений в рамках применимого законодательства
- Использование в соответствии с выбранным тарифом

### 4.2. Запрещено

- Использование Сервиса для незаконной деятельности
- Распространение ключей доступа третьим лицам
- Попытки взлома, обхода ограничений или злоупотребления инфраструктурой
- Коммерческое перепродажа доступа без согласия Сервиса
- Использование для рассылки спама или вредоносной деятельности

### 4.3. Последствия нарушений

При выявлении нарушений Сервис вправе:

- Заблокировать аккаунт пользователя
- Аннулировать ключи доступа
- Отказать в обслуживании без предварительного уведомления

## 5. Роли и доступы (для администраторов)

| Роль | Возможности |
|------|-------------|
| **Владелец** | Полный доступ ко всем функциям панели |
| **Технический администратор** | Управление пользователями, устройствами, серверами |
| **Менеджер** | Просмотр пользователей, платежи, поддержка |

Вход в панель управления: логин/пароль + Telegram-код (5-минутные коды через control-бот).

---

## 6. Ограничение ответственности

- Сервис предоставляется **«как есть»** (as is)
- Сервис **не гарантирует** бесперебойную работу, особенно в условиях блокировок
- Сервис **не несёт ответственности** за действия пользователей, нарушающие применимое законодательство
- Максимальная ответственность Сервиса ограничена суммой оплаченных пользователем услуг за текущий расчётный период

---

## 7. VPN-ноды и инфраструктура

| Нода | Runtime | Режимы |
|------|---------|--------|
| Германия | 3x-ui, VLESS + Reality + TCP | Стабильный / Мобильный / Резерв |
| Дания | Xray core, VLESS + Reality + XHTTP | Стабильный / Мобильный / Резерв |
| Эстония | VLESS + Reality + TCP | Только unified-link/feed (резерв) |

Сервис оставляет за собой право изменять конфигурацию нод без предварительного уведомления.

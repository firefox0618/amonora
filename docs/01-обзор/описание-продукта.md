# 📖 Описание продукта Amonora

## Что такое Amonora

**Amonora** — это экосистема цифровых сервисов для безопасного и приватного доступа в интернет. 

### Основной продукт: VPN-сервис

Весь путь пользователя построен вокруг **Telegram**:
1. Человек приходит через сайт или сразу в бота
2. Получает пробный доступ (3 дня)
3. Подключает устройство, выбирает страну и протокол
4. Управляет подпиской прямо в Telegram

### Ключевая ценность

- ✅ **Простота** — не нужно разбираться в технологиях, всё работает через Telegram
- ✅ **Быстрый старт** — пробный период 3 дня без оплаты
- ✅ **Мультиплатформенность** — работает на всех устройствах
- ✅ **Прозрачность** — видно баланс, сроки подписки, историю платежей
- ✅ **Поддержка** — помощь через бота поддержки 24/7

### Источники дохода

1. **Подписки** — ежемесячные/квартальные/годовые платежи
2. **Дополнительные слоты** — оплата за дополнительные устройства
3. **Реферальная программа** — органический рост через рекомендации

# Shared Transitional Parts

Временный переходный каталог для частей, которые на этапе разделения могут использовать и `site`, и `client`.

Примеры того, что может временно оказаться здесь:

- общий bootstrap FastAPI-приложений;
- общая настройка шаблонов и статики;
- общие middleware и утилиты публичных поверхностей;
- общие функции рендеринга и canonical URL.

Это не конечный продуктовый контур.

Цель каталога — не копировать код при разделении `landing`, а временно держать общее в одном месте до дальнейшего разнесения.

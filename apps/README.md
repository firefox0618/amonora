# Apps

Этот каталог вводится как целевая точка для отдельных приложений проекта.

На текущем этапе здесь создаётся только безопасный каркас без переноса runtime-логики.

Первичные контуры:

- `site/` — будущий публичный сайт;
- `client/` — будущая клиентская публичная поверхность;
- `shared/` — временные общие части переходного периода между `site` и `client`.

Пока действующие runtime-entrypoint'ы остаются в текущих каталогах проекта.

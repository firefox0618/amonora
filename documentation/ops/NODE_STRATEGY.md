# 2. ХАРАКТЕРИСТИКИ

## Минимум

- CPU: 2 vCPU
- RAM: 2 GB
- Network: 1 Gbps shared

---

## Оптимум

- CPU: 4 vCPU
- RAM: 4–8 GB
- Network: 1–10 Gbps

---

# 3. ЛОКАЦИИ

---

# 4. ПРОВАЙДЕРЫ

Искать:
- не перегруженные ASN
- нормальная репутация IP
- без массовых VPN клиентов

---

# 5. ЧТО ВАЖНО

- CPU важнее диска
- latency важнее “гигабитов”
- стабильность важнее цены

---

# 6. КОГДА ДОБАВЛЯТЬ НОДУ

- CPU > 70%
- жалобы на скорость
- рост пользователей

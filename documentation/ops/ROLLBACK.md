# ЦЕЛЬ

# 1. BACKUP ПЕРЕД ЛЮБЫМ ИЗМЕНЕНИЕМ

## База данных

- [ ] Сделать dump PostgreSQL

Команда:
```
pg_dump -U amonora -h localhost amonora_db > backup.sql
```

---

## ENV ФАЙЛЫ

Сохранить:
- [ ] .env backend
- [ ] dashboard env
- [ ] bot env

---

## NGINX

- [ ] сохранить конфиг
```
/etc/nginx/sites-enabled/*
```

---

## SYSTEMD

- [ ] сохранить сервисы
```
/etc/systemd/system/*
```

---

## VPN (КРИТИЧНО)

- [ ] сохранить inbound Germany
- [ ] сохранить inbound Estonia
- [ ] сохранить:
  - ports
  - SNI
  - shortId
  - keys

---

# 2. БЫСТРЫЙ ОТКАТ

## Если сломался VPN

- [ ] вернуть старый inbound
- [ ] перезапустить xray / 3x-ui

---

## Если сломался backend

- [ ] перезапустить сервисы
```
systemctl restart amonora*
```

---

## Если сломался dashboard

- [ ] проверить nginx
- [ ] проверить порт 3001
- [ ] перезапустить

# 3. ПРОВЕРКА ПОСЛЕ ОТКАТА

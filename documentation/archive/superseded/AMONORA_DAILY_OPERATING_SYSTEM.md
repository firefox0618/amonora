# 🧠 ГЛАВНЫЙ ПРИНЦИП

> Каждый день = контроль + развитие + деньги

# ⏰ 1. УТРО (START OF DAY) — 20–40 МИН

## Цель:
понять состояние системы

### Проверить:

- [ ] VPN работает?
- [ ] есть ли жалобы?
- [ ] backend жив?
- [ ] dashboard работает?
- [ ] есть ли ошибки в логах?
- [ ] есть ли новые support тикеты?
- [ ] есть ли платежи?

---

## Быстрые метрики:

- [ ] активные пользователи
- [ ] активные устройства
- [ ] ноды healthy?
- [ ] есть ли деградации?

---

## Итог:

👉 понять:
- всё стабильно
- или есть проблема

---

# 🔧 2. БЛОК СТАБИЛЬНОСТИ (CORE WORK) — 1–2 ЧАСА

## Если есть проблемы:

- [ ] исправить критичные баги
- [ ] не трогать лишнее
- [ ] записать в changelog
- [ ] проверить после фикса

---

## Если всё стабильно:

- [ ] мелкие улучшения
- [ ] фиксы UX
- [ ] улучшение логики

---

# 🚀 3. БЛОК РАЗВИТИЯ (BUILD) — 2–4 ЧАСА

Фокус:

👉 ОДНА задача

---

## Возможные задачи:

### VPN
- новый inbound
- улучшение логики выдачи
- стабильность

### Automation
- бот
- workflow
- кейс

### Backend
- модульность
- API
- логика

## Правило:

❌ не делать 5 задач  
✔ делать 1 задачу до результата

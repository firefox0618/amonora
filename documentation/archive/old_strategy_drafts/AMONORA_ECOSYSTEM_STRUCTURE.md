- админ-панель

- Telegram Bot
- Backend API
- PostgreSQL
- Dashboard
- Support system

## ЦЕЛЬ:

→ централизованное управление всей системой

### ЧТО ВКЛЮЧАЕТ:

### ФУНКЦИИ:

### ЦЕЛЬ:

- приходит уведомление

### ЧТО ВКЛЮЧАЕТ:
